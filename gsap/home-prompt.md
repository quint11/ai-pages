# 企业官网首页 prompt
```
设计一个现代感强、视觉吸引力高的品牌官网首页动画效果，使用 GSAP 实现以下动效细节：

1. 页面加载动画：
   - Logo 从中间渐显放大进入，带有 ease-out 的缩放弹性效果
   - 背景图淡入，从透明到不透明（opacity 0 → 1），伴随微弱缩放（scale 1.05 → 1）
   - 主标题文字逐字母淡入 + 向上浮动（每个字母延迟 0.05s）

2. 导航栏动效：
   - 加载完成后顶部导航栏从上方向下滑入（y: -50 → 0, opacity: 0 → 1）
   - 鼠标悬停时，菜单项下划线从左向右展开（使用 transform scaleX 动效）

3. 滚动触发动画（使用 ScrollTrigger）：
   - 向下滚动时，每个模块卡片依次浮现，y轴上移 + 渐显（y: 30 → 0, opacity: 0 → 1）
   - 背景图元素带有视差效果（parallax），随滚动略微移动（yPercent: 0 → -10）

4. CTA（Call to Action）按钮动效：
   - 悬停时按钮轻微放大 + 发光（scale 1 → 1.05, box-shadow 增强）
   - 点击时有“按压回弹”动画（scale 1.05 → 0.95 → 1）

5. 响应式优化：
   - 所有动画需在移动端设备上流畅展示，避免性能卡顿
   - 动效应适配不同分辨率，文字和按钮位置自适应调整

整体风格参考 Apple、Stripe、Tesla 官网的高级动画动效，动画节奏自然、有呼吸感，视觉层次分明，提升品牌调性。
```
